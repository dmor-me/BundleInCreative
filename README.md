# Bundle In Creative Mod for Minecraft 1.20.1 (Forge)

This mod adds the vanilla Bundle item to the Tools & Utilities creative tab, making it visible in:
- Creative Menu
- EMI/JEI/REI recipe viewers
- FTB Quests
